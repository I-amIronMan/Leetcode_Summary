#### 关于子数组的常见问题
子数组指的是数组中的连续部分，和字符串的子串类似。  
###### 常见解法
1. 动态规划  
一般子数组的题使用动态规划求解时，$ dp[i] $表示必须以i位置结尾的状态。  
2. 滑动窗口  
通常情况下使用双指针维持窗口的左右边界，窗口只能从左往右滑动；特殊情况下需要使用单调队列维持窗口的左右边界。    
3. 单调栈  
一般题目中提到了最小值或最大值则可能会用单调栈的方法求解。  
###### 常用技巧
1. 前缀和预处理技巧  
当需要求数组中子数组的和时，首先生成一个前缀和数组，然后再计算子数组累积和时就可以在O(1)的时间计算出来。
